System Architecture

โครงสร้างสถาปัตยกรรมระบบทางเทคนิค

ออกแบบภายใต้มาตรฐาน Cloud-Native & Serverless Microservices เพื่อความปลอดภัย ความเร็วสูง และความพร้อมใช้งาน 99.9%

🌐
1. Presentation Layer
Frontend Web Portal & Studio
หน้าอินเทอร์เฟซสำหรับผู้ยื่นคำร้อง และห้องทำงาน AI Blueprint Studio สำหรับวิศวกรที่มาพร้อมระบบ Drag & Zoom สไตล์ Photoshop
Firebase Hosting CDN HTML5 / Vanilla CSS Responsive Glassmorphism
⚡
2. API Gateway & Logic
Cloud REST API Backend
ศูนย์กลางประมวลผลคำร้องจากแบบฟอร์ม ตรวจสอบความถูกต้องของข้อมูล (Validation) และจัดการเส้นทางคำสั่งเชื่อมต่อไปยัง AI และฐานข้อมูล
Render Cloud Node.js Express Microservices CORS & Security Shield
🧠
3. AI Intelligence Layer
Multimodal Vision Inspection
สมองกล AI อัจฉริยะที่สามารถอ่านไฟล์แปลน (PDF/IMG) วิเคราะห์ระยะร่น ทางหนีไฟ ความปลอดภัยโครงสร้าง และกฎหมายอาคารได้ในไม่กี่วินาที
Google Gemini Multimodal AI Vision & Reasoning Engine Structured Engineering Prompt
🗄️
4. Storage & Email Sync
Real-time DB & Notification
ระบบบันทึกข้อมูลคำร้องลงตารางคลาวด์แบบ Real-time และระบบแจ้งเตือนอัตโนมัติผ่านอีเมล 2 ช่องทาง ทั้งถึงผู้ขอและทีมวิศวกรผู้ดูแลระบบ
Google Sheets Cloud DB Google Drive Blueprint Storage Nodemailer SMTP Dispatch Engine
End-to-End Workflow

ขั้นตอนการทำงานตั้งแต่ผู้ยื่นคำร้องถึงวิศวกร

กระบวนการพิจารณาตรวจสอบแบบแปลนที่รวดเร็ว โปร่งใส และประสานงานอย่างไร้รอยต่อ

1
🏢 ยื่นคำร้องผ่านเว็บ
ผู้ยื่นคำร้องหรือผู้รับเหมากรอกแบบฟอร์มตรวจสอบ/คัดลอกแปลน พร้อมแนบลิงก์ไฟล์แปลนและระบุรายละเอียดคำอธิบายเพิ่มเติมทางออนไลน์
2
⚡ ซิงค์ข้อมูล & แจ้งเตือน
ระบบบันทึกข้อมูลลง Google Sheets ทันที พร้อมส่งอีเมลยืนยันการรับเรื่องถึงผู้ยื่นคำร้อง และส่งอีเมลแจ้งเตือนมีคำร้องใหม่ถึงวิศวกรทันที
3
✋ ตรวจแปลนด้วย Studio
วิศวกรเปิดหน้าต่าง AI Blueprint Studio ใช้ปุ่มซูมสีขาวคมชัด และเครื่องมือ Hand Tool (`✋`) ลากดูรายละเอียดแปลนได้รอบทิศทาง
4
🤖 วิเคราะห์เชิงลึกด้วย AI
กดเริ่มวิเคราะห์ด้วย Gemini AI เพื่อประเมินข้อดี จุดควรระวัง และกฎหมายอาคาร จากนั้นวิศวกรพิจารณาอนุมัติหรือขอเอกสารเพิ่ม
5
💌 แจ้งผลลัพธ์เป็นทางการ
ระบบจัดทำรายงานสรุปผลการพิจารณา และจัดส่งอีเมลแจ้งข้อสั่งการหรืออนุมัติคัดลอกแบบแปลนถึงผู้ยื่นคำร้องอย่างรวดเร็วและแม่นยำ
Key Highlights

6 จุดเด่นสำคัญของโปรเจค STeP CMU Blueprint Portal

นวัตกรรมที่ช่วยยกระดับการให้บริการของอุทยานวิทยาศาสตร์และเทคโนโลยี มหาวิทยาลัยเชียงใหม่

🤖
1. AI Multimodal Vision Engine
ผสานรวมเทคโนโลยี Google Gemini AI ในการอ่านและตีความแบบแปลนวิศวกรรม 2D ช่วยตรวจสอบความปลอดภัยและกฎหมายอาคารอัตโนมัติ ลดเวลาทำงานของวิศวกรลงกว่า 70%
✋
2. Photoshop-Style Hand Tool Studio
หน้าต่างตรวจสอบแปลนที่ให้ประสบการณ์เหมือนโปรแกรม CAD และ Photoshop: ลากย้ายหน้าต่างได้อิสระ (`Draggable`) พร้อมปุ่มซูมสีขาวเด่นชัด และเครื่องมือ Hand Tool (`✋`) สำหรับคลิกค้างแล้วลากดูแปลนรอบทิศทาง
📊
3. Real-time Google Sheets Cloud DB
เชื่อมต่อข้อมูลทุกคำร้องเข้ากับ Google Sheets อัตโนมัติในเสี้ยววินาที ทำให้ทีมวิศวกรสามารถค้นหา จัดกลุ่ม ออกสถิติ และทำงานร่วมกัน (`Collaboration`) ได้ทันทีโดยไม่ต้องติดตั้งโปรแกรมเพิ่ม
💌
4. Automated Dual Email Dispatcher
ระบบแจ้งเตือนผ่านอีเมล 2 ทิศทางอัตโนมัติ: ทั้งการแจ้งยืนยันรับคำร้องถึงผู้ยื่นคำร้อง และอีเมลแจ้งเตือนด่วนถึงทีมวิศวกร ทำให้ทุกขั้นตอนโปร่งใส ติดตามได้ และไม่มีคำร้องตกหล่น
🎨
5. High-Contrast Ergonomic UI
ออกแบบด้วยสไตล์ Modern Dark Glassmorphism พร้อมคู่สีโทนสว่างที่อ่านง่าย (`High-Contrast Buttons & Tokens`) ช่วยลดความเมื่อยล้าของสายตาวิศวกรเมื่อต้องพิจารณาแบบแปลนที่มีรายละเอียดสูงต่อเนื่อง
🛡️
6. Enterprise Serverless Scalability
ทำงานบนคลาวด์มาตรฐานระดับโลก Firebase Hosting CDN และ Render Cloud Backend รองรับผู้ใช้งานจำนวนมากพร้อมกันได้อย่างเสถียร ปลอดภัย และพร้อมขยายบริการในอนาคตอย่างไร้ขีดจำกัด
